🛠️ Bash Scripting Suite for System Maintenance

A complete automation suite built using Bash shell scripting to simplify essential Linux system maintenance tasks.
This project includes backup, update, cleanup, and log monitoring utilities — all accessible through a menu-driven interface.


📌 Features
✔️ 1. Backup Script (backup.sh)

. Creates compressed .tar.gz backup

. Stores backups with timestamps

. Logs all activities

✔️ 2. Update & Cleanup Script (update_cleanup.sh)

. Updates package lists

. Upgrades system

. Removes unnecessary files

. Logs actions for tracking

✔️ 3. Log Monitor Script (log_monitor.sh)

. Scans log files

. Detects errors, warnings, failures

. Records findings in monitor log

✔️ 4. Menu-driven Interface (maintenance_suite.sh)

. Simple UI

. One-click execution

. Beginner-friendly


🔧 Technologies & Tools

Operating System: Ubuntu / Linux

Language: Bash Shell Script

Tools: Terminal

Utilities: apt-get, tar, mkdir, grep, tee

Concepts: Automation, File Handling, Logging, Menu-Driven Programming


🧪 How It Works

🔹 Backup

Creates a compressed backup of the user’s Documents folder.

🔹 Update & Cleanup

Updates system packages and removes unnecessary data.

🔹 Log Monitor

Scans log files for issues.

🔹 Menu Interface

Allows users to choose any operation easily.


📈 Results & Observations

. Backup created successfully

. Cleanup and update executed properly

. Log monitoring works accurately

. Menu interface improves usability

. Scripts are modular & reusable

. All actions are logged for safety
